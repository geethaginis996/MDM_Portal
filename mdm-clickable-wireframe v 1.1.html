Wireframe Guide

Everything in this wireframe is clickable. Try these paths:

  • Side nav → jump between modules; click section headers to expand/collapse
  • button at top of sidebar collapses to icon-only mode (or press [)
  • Field Master → click any row → object page with 6 tabs; Value Help and Validation tabs use dropdowns populated from the registries
  • Validation Rules → click any row → full detail page (3 tabs: General / Input Parameters / Linked Fields)
  • Value Tables → click any row → full detail page (4 tabs: General / Input Parameters / Output Columns / Usage)
  • Field Groups → unified list of main & sub groups → click any row → detail page with Sub Groups and Assigned Fields tabs
  • BP Roles → FLVN01 → Field Assignment → "Assign Fields" dialog
  • BP Roles → CRM000 → Prerequisite Fields tab → "See runtime preview"
  • Create BP → pick a role → tabs are generated from its field config; prereq tab must be filled first to unlock others
Note: This is a wireframe, not a live system. Data doesn't persist across navigation.
